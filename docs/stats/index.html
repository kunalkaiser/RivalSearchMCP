Repository Stats

Daily snapshots of public GitHub repository activity for damionrashford/RivalSearchMCP — captured automatically each day and stored in git.

View the full report  → Updated daily · 23:00 UTC

What this is not. This page does not track visitors — no analytics, no cookies, no third-party scripts or images. It also does not observe MCP server usage in any way. The only data collected is the public, repository-level activity that GitHub itself records on its own Traffic API (views, clones, referrers, stars, forks).

What gets captured

  • Unique and total views per day
  • Unique and total clones per day
  • Top referrers — where landings come from
  • Top paths — which files inside the repo are viewed
  • Stargazer and fork counts over time

Where the data lives

The action jgehrcke/github-repo-stats runs daily and commits raw CSVs plus a generated HTML and PDF report to the github-repo-stats branch of this repository. No external database, no cloud storage — just git history.
